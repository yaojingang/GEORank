# 08. 迁移工作流

## 总体顺序

1. 基线冻结
2. 仓库骨架
3. API 契约冻结
4. SDK 生成
5. 前台公开页迁移
6. 鉴权与功能页迁移
7. 后台迁移
8. 多语言模型迁移
9. CLI 与运维命令
